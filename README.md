# Sound-Assistant-Bot

This is a sound Assistant Bot. Following Functionalities are added-
1. Wikipedia Search
2. Open Google,Youtube 
3. Send Emails
4. Time Assistant
5. Open music(spotify app)
6. Prepare Codeforces contest(open codeforces ad an online ide)
7. Change Speaker
8. Shut down and Restart
